## [1.0.0] - 2026-06-08

### Added
- File type association and syntax highlighting for `.xcp` process definition files
- Comment toggling and brace matching
- Go to Declaration with multi-target popup showing line number and code preview
- Find Usages and Highlight Usages for variables, phases, and functions
- Reference resolution for variables, phases, and JS functions across the file
- Completion contributor for XCP keywords and known symbols
- Schema validation inspection with warnings for unknown or missing fields
